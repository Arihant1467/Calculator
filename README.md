# Calculator
A Calculator app using React JS and Node JS

# Start the Project

#### Start the frontend
```
cd Calculator/calcualtor-frontend/
npm install
npm start
```

#### Start the backend
```
cd Calculator/calculator-backend
npm install
node app.js
```
